# aws-rekognition-ruby-example
Example using aws rekognition via AWS Ruby SDK

# .env FILE
Please put `.env` file in the project root directory.

```.env
AWS_KEY=[YOUR_API_KEY]
AWS_SECRET=[YOUR_API_SECRET]
AWS_REGION=[YOUR_API_REGION]
```
